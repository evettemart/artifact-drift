# Bob Plan Mode Rules For Hackathon Tools

Use Plan mode before larger edits or when the team is deciding scope.

- Read `.hackathon/epic.md` before planning implementation work.
- If `.hackathon/epic.md` is missing, recommend `/setup-hackathon`.
- Keep plans small enough for hackathon execution.
- Tie each planned task to a story, Definition of Done item, or Review Checklist item.
- Identify evidence the team should create for reviewers.
- Do not plan organizer-only review, scoring, awards, or chronicle work.
