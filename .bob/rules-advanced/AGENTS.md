# Bob Advanced Mode Rules For Hackathon Tools

Use Advanced mode only when deeper tool access is needed.

- Be careful with MCP, browser, terminal, and filesystem access.
- Do not connect tools that expose secrets, customer data, or unrelated personal files.
- Before running broad commands, explain what will be inspected or changed.
- Do not run destructive commands.
- Use `.hackathon-tools/scripts/` or `.bob/scripts/` helpers when generating repo summaries or validating JSON.
- Keep final participant outputs score-free and review-focused.
